# Quick Start: Setting Up Python in VS Code (Windows Guide)

This guide will walk you through installing Python on Windows and setting it up in Visual Studio Code (VS Code) so you can run your practice scripts.

---

## Step 1: Download and Install Python

Before using VS Code, you need to install Python on your Windows system.

1. Go to the official [Python Downloads Website](https://www.python.org/downloads/).
2. Click the yellow **Download Python** button to get the latest Windows installer.
3. Open the downloaded `.exe` installer file.
4. **CRITICAL STEP:** At the bottom of the installer window, check the box that says **"Add python.exe to PATH"**. *If you miss this step, VS Code won't be able to find Python!*
5. Click **Install Now** at the top of the window.
6. Once finished, click **Close**.

---

## Step 2: Install VS Code

If you don't have Visual Studio Code installed yet:

1. Go to the [VS Code Website](https://code.visualstudio.com/).
2. Click the **Download for Windows** button.
3. Run the downloaded installer and accept the agreement.
4. Keep the default settings and click **Next** until the installation completes, then click **Finish** to open VS Code.

---

## Step 3: Install the Python Extension in VS Code

VS Code needs an extension to run and highlight your Python code.

1. Open **VS Code**.
2. Click the **Extensions** icon on the left-hand sidebar (it looks like four squares, or you can press `Ctrl + Shift + X`).
3. In the search bar at the top, type `Python`.
4. Find the extension labeled **Python** (published by **Microsoft**) and click the blue **Install** button.

---

## Step 4: Run Your Practice File!

Now you are ready to open and run your `.py` files.

1. In VS Code, go to **File** > **Open File...** (or press `Ctrl + O`).
2. Choose your practice script (for example, `variables_and_types_only.py`).
3. Look at the **top-right corner** of the screen. You should see a small **Play button** (a triangle pointing right).
4. Click that **Play button**. 
5. A terminal window will open at the bottom of your screen, and you'll see your code run!
